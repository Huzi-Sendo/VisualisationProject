Section 01 · Spec 1.1

SI Units, Quantities and Measurements

1.1 · SI Base Units

The seven SI base units are: kilogram (kg), metre (m), second (s), ampere (A), kelvin (K), mole (mol), candela (cd). All other units are derived. The MCQ presents a mixed list and asks you to identify which is a base unit.

Exam method: SI base unit MCQ

  1. Memorise the seven: kg, m, s, A, K, mol, cd.
  2. Reject compound or named derived units: coulomb $= \mathrm{A\,s}$, ohm $= \mathrm{kg\,m^2\,s^{-3}\,A^{-2}}$, volt $= \mathrm{kg\,m^2\,s^{-3}\,A^{-1}}$.
The ampere IS a base unit. The coulomb is NOT.

1.2 · Percentage Uncertainty

Percentage uncertainty $$\%\text{unc} = \frac{\Delta Q}{Q} \times 100$$ $$\text{For } W = V \cdot I \cdot t:\quad \%W = \%V + \%I + \%t$$ $$\text{For } Q^n:\quad \%Q^n = n \times \%Q$$

Exam method: W = VIt uncertainty (2024)

  1. $\%V = \frac{0.20}{12} \times 100 = 1.67\%$
  2. $\%I = \frac{1.0}{80} \times 100 = 1.25\%$
  3. $\%t = \frac{0.01}{60} \times 100 = 0.017\%$
  4. Total $= 1.67 + 1.25 + 0.017 = 2.9\%$
Never add absolute uncertainties. Always convert mA to A before calculating $\%I$.

Exam method: consistency check — % difference vs % uncertainty (2023 Q21c-iii)

  1. Calculate $\%\text{diff} = \dfrac{|E_\text{calc} - E_\text{ref}|}{E_\text{ref}} \times 100$
  2. Example: $\dfrac{|1.15 - 1.17|}{1.17} \times 100 = 1.7\%$
  3. Since $1.7\% < 3.3\%$ (the calculated $\%$ uncertainty), the values are consistent.
Always state the comparison explicitly in words: "% difference < % uncertainty, therefore consistent."

Section 02 · Spec 3.1

Forces and Motion

2.1 · Terminal Velocity

At terminal velocity $$W = F_\text{drag} + F_\text{upthrust} \quad\Rightarrow\quad F_\text{resultant} = 0 \quad\Rightarrow\quad a = 0 \quad\Rightarrow\quad v = \text{constant}$$

Exam method: explain terminal velocity (3 marks)

  1. Mark 1: weight = air resistance (+ upthrust).
  2. Mark 2: resultant/net force $= 0\ \mathrm{N}$.
  3. Mark 3: acceleration $= 0$, so velocity is constant.
"Forces are balanced" alone scores 0. Never say velocity decreases at terminal velocity.

2.2 · Kinematics and Free Fall

SUVAT equations $$v = u + at \qquad s = ut + \tfrac{1}{2}at^2 \qquad v^2 = u^2 + 2as \qquad s = \tfrac{(u+v)t}{2}$$ $$\text{Free fall from rest: } u=0,\ a = g = 9.81\ \mathrm{m\,s^{-2}} \qquad v = \sqrt{2gh}$$ $$\text{Finding } g \text{ from timing: } g = \frac{2s}{t^2}$$

Exam method: calculate g from drop-time data (3 marks, 2024)

  1. Inspect readings. If one is clearly anomalous, state it is anomalous and exclude it — OCR requires explicit identification before exclusion.
  2. Calculate mean $t$ from remaining values.
  3. Substitute into $g = \dfrac{2s}{t^2}$ and evaluate.
Silently ignoring an anomalous value loses the mark. You must state it is anomalous first.

2.3 · Impulse and Momentum (2023 Q24)

Impulse and change in momentum $$p = mv \qquad \text{Impulse} = F\Delta t = \Delta p = m(v_2 - v_1)$$ $$F = \frac{\Delta p}{\Delta t} = \frac{m(v_2 - v_1)}{\Delta t}$$ $$\text{Projectile (no air resistance): } x = v_x \cdot t$$
Sign rule for rebounds: if the ball reverses direction, $v_1$ and $v_2$ have opposite signs, so $\Delta p = m(v_2 - v_1)$ effectively adds the speeds: $\Delta p = m(7.2 + 3.6) = 0.058 \times 10.8 = 0.63\ \mathrm{kg\,m\,s^{-1}}$.

Exam method: force from collision with wall (3 marks, 2023 Q24a)

  1. $\Delta p = m(v_2 - v_1) = 0.058 \times (7.2 - (-3.6)) = 0.058 \times 10.8 = 0.626\ \mathrm{kg\,m\,s^{-1}}$
  2. $F = \dfrac{\Delta p}{\Delta t} = \dfrac{0.626}{0.052} = 12\ \mathrm{N}$
Forgetting the sign change on the rebound velocity gives $\Delta p = 0.058 \times 3.6$ — only half the correct value.

Exam method: projectile lands-in-ring check (3 marks, 2023 Q24b)

  1. Horizontal distance $= v_\text{rebound} \times t = 3.6 \times 0.58 = 2.09\ \mathrm{m}$
  2. Target range: centre $\pm$ radius $= 2.0 \pm 0.15\ \mathrm{m}$, so acceptable range is $1.85\ \mathrm{m}$ to $2.15\ \mathrm{m}$.
  3. State: $1.85 < 2.09 < 2.15$, therefore the ball lands in the ring.
You must state the numerical comparison explicitly. Calculating 2.09 m alone is not enough for full marks.

2.4 · Stokes' Law — Terminal Velocity in Fluids (2023 Q25)

Stokes terminal velocity $$v = \frac{2r^2 g(\rho_s - \rho_w)}{9\eta}$$ $$r = \sqrt{\frac{9\eta v}{2g(\rho_s - \rho_w)}}$$

$r$ = radius (m), $\rho_s$ = particle density, $\rho_w$ = fluid density (kg m$^{-3}$), $\eta$ = viscosity (Pa s)

Exam method: show terminal velocity from displacement data (2 marks, 2023 Q25c)

  1. Use $v = s/t$ for two adjacent 0.1 s intervals from 0.2 s onwards (not from 0.0 s where particle is still accelerating).
  2. Example: $0.2\to0.3$ s: distance $= 2.7\ \mathrm{cm}$, so $v = 0.027/0.1 = 0.27\ \mathrm{m\,s^{-1}}$. Repeated interval gives same value → terminal velocity confirmed.

Exam method: identify soil type from terminal velocity (3 marks, 2023 Q25d)

  1. $r = \sqrt{\dfrac{9 \times 1.0\times10^{-3} \times 0.27}{2 \times 9.81 \times (1500-1000)}} = \sqrt{2.48\times10^{-7}} = 5.0\times10^{-4}\ \mathrm{m}$
  2. Diameter $= 2r = 1.0\times10^{-3}\ \mathrm{m} = 1.0\ \mathrm{mm}$
  3. $0.05\ \mathrm{mm} < 1.0\ \mathrm{mm} < 2.0\ \mathrm{mm}$ → sand.
Calculate diameter (not radius) before comparing to the table. The formula gives $r$; you must double it.

2.5 · Equilibrium and Moments

Moments and equilibrium $$\text{Moment} = F \times d_\perp \qquad P = Fv = F\cos\theta \cdot v\ \text{(angled force)}$$ $$\text{Hanging picture: } W = 2T\cos\theta \quad (\theta\ \text{from vertical})$$ $$\text{Equilibrium: } \sum F = 0 \text{ and } \sum \text{moments} = 0$$
Two-force equilibrium MCQ (2023 Q16): Only statement 1 is correct (forces are equal and opposite). Statement 2 is false — Newton's 3rd law pairs act on different bodies. Statement 3 is false — the object could be moving at constant velocity.

Section 03 · Spec 3.2

Work, Energy and Materials

3.1 · Young Modulus (2023 Q21)

Stress, strain and Young modulus $$\sigma = \frac{F}{A} \qquad \varepsilon = \frac{x}{L} \qquad E = \frac{\sigma}{\varepsilon} = \frac{FL}{Ax}$$ $$\text{Since } F = kx: \quad E = \frac{kx \cdot L}{A \cdot x} = \frac{kL}{A}$$ $$\text{From } F\text{-}x \text{ graph:}\quad k = \text{gradient} = \frac{\Delta F}{\Delta x}$$

Exam method: measure cross-sectional area of wire (3 marks, 2023 Q21b)

  1. Use a micrometer (or callipers) to measure the diameter.
  2. Measure the diameter in different places along the wire and calculate a mean.
  3. Calculate $A = \pi r^2$ where $r = d/2$, or equivalently $A = \pi d^2/4$.
Must measure in multiple places — wires are not perfectly uniform. One measurement loses mark 2.

Exam method: derive E = kL/A from first principles (2 marks, 2023 Q21c-i)

  1. State all four equations: $E = \sigma/\varepsilon$, $F = kx$, $\sigma = F/A$, $\varepsilon = x/L$
  2. Substitute clearly: $E = \dfrac{F/A}{x/L} = \dfrac{kx/A}{x/L} = \dfrac{kL}{A}$
Do not just quote the result. OCR requires you to show all four equations and the algebraic steps.

3.2 · Springs and Elastic PE

Hooke's law and elastic energy $$F = kx \qquad E_\text{elastic} = \tfrac{1}{2}kx^2 \qquad \text{Area under } F\text{-}x \text{ graph} = E_\text{elastic}$$

Both compression and extension increase stored energy. MCQ: $k = 4900\ \mathrm{N\,m^{-1}}$, $x = 0.50\ \mathrm{m}$: $E = \frac{1}{2}(4900)(0.25) = 612.5 \approx 610\ \mathrm{J}$ increase.

3.3 · Stress-Strain Graphs

Brittle: linear stress-strain graph that terminates abruptly — no plastic region, fractures at the elastic limit. Ductile: linear elastic region followed by a large plastic region. The MCQ: select the graph that is linear and ends without curving.

Section 04 · Spec 4.1

Electricity

4.1 · Resistance and Resistivity

Resistance and resistivity $$R = \frac{\rho L}{A} \qquad \text{Gradient of } R\text{-}L \text{ graph} = \frac{\rho}{A}$$ $$I = nAve \quad\Rightarrow\quad v_d = \frac{I}{nAe}$$

$n$ = number density (material property — same for all wires of same material), $e = 1.6\times10^{-19}\ \mathrm{C}$

Exam method: two copper wires in series — which quantities are equal? (2023 Q9)

  1. Wires in series → same current $I$ through both.
  2. Same material (copper) → same $n$ (number density) for both.
  3. Different area $A$ → different drift velocity ($v_d = I/nAe$) and different resistance ($R = \rho L/A$).
Answer A: charge carrier density and current are both equal. Drift velocity and resistance are NOT equal.
Resistance vs resistivity when length increases (2023 Q10): $R = \rho L/A$ so longer wire → larger $R$. Resistivity $\rho$ is a material property — it does not change when you change the geometry. Answer: C (resistance increases, resistivity constant).

4.2 · EMF and Internal Resistance

EMF equations $$\varepsilon = I(R + r) = IR + Ir \qquad V_\text{terminal} = \varepsilon - Ir$$ $$\text{Cells in series: total EMF} = n\varepsilon,\quad \text{total } r = nr$$ $$\text{Power to load: } P = IV = I(\varepsilon - Ir)$$ $$\text{EMF} = \frac{\text{energy from chemical source}}{\text{charge}}\ (\mathrm{J\,C^{-1}})$$

Exam method: find internal resistance from circuit (2023 Q7)

  1. Two cells in series: total EMF $= 2 \times 3.0 = 6.0\ \mathrm{V}$, total $r = 2r$.
  2. Apply $\varepsilon_\text{total} = I(R + r_\text{total})$: $\quad 6.0 = 0.040(145 + 2r)$
  3. $150 = 145 + 2r \Rightarrow r = 2.5\ \Omega$ per cell. Answer: C.
Use total EMF from both cells, not just one. Internal resistances add in series.

Exam method: show P = 3I(ε − Ir) for three cells in series (3 marks, 2024)

  1. Mark 1 (B): state $\varepsilon = Ir + IR$.
  2. Mark 2 (M): three cells → total EMF $= 3\varepsilon$, total $r = 3r$: $\quad 3\varepsilon = 3Ir + IR$
  3. Mark 3 (A): $P = I^2R = I(3\varepsilon - 3Ir) = 3I(\varepsilon - Ir)$. Show every step.
Never skip steps in a "show that" question. Every algebraic line must be visible.

4.3 · Potential Dividers and LED Circuits

Potential divider and LED threshold $$V_\text{out} = \frac{R_2}{R_1 + R_2} \cdot V_\text{supply}$$ $$eV_0 = hf \quad\Rightarrow\quad V_0 = \frac{hf}{e}$$ $$\text{Green LED: higher } f \Rightarrow \text{higher } V_0 \Rightarrow V_0 \text{ increases, wavelength decreases (2023 Q12, Answer D)}$$

Exam method: thermistor-LED switch-on (4 marks, 2024)

  1. Read LED switch-on voltage from I-V graph (e.g. $3.0\ \mathrm{V}$).
  2. Read $R_\text{therm}$ at the threshold temperature from R-$\theta$ graph (e.g. $30\ \Omega$ at $30\,^\circ\mathrm{C}$).
  3. $V_\text{LED} = \dfrac{55}{55+30} \times 5 = 3.24\ \mathrm{V}$
  4. State: $3.24\ \mathrm{V} > 3.0\ \mathrm{V}$, therefore LED switches on.
The numerical comparison must be written explicitly. Calculating 3.24 V alone does not earn full marks.

4.4 · Charge Counting (2023 Q20)

Charge and electron count $$Q = ne \quad\Rightarrow\quad n = \frac{Q}{e} = \frac{4.0\times10^{-18}}{1.6\times10^{-19}} = 25\ \text{electrons lost}$$ $$\text{Gaining positive charge} \Leftrightarrow \text{losing electrons. Answer: C.}$$

Section 05 · Spec 5.1 and 5.2

Waves

5.1 · Wave Properties and Speed

Wave equations $$v = f\lambda \qquad T = \frac{1}{f}$$ $$\text{Deep to shallow water: speed decreases, frequency constant, wavelength decreases. (2023 Q17, Answer B)}$$

5.2 · Longitudinal Waves (2023 Q22a)

Exam method: explain compressions and rarefactions (3 marks)

  1. Molecules vibrate/oscillate.
  2. Oscillations are parallel to the direction of energy transfer/propagation.
  3. This creates regions of high and low pressure (compressions and rarefactions).
Do not reference varying amplitude. Do not say oscillations are perpendicular.

5.3 · Stationary Waves (2023 Q22b)

Stationary wave relationships $$\text{Node spacing} = \frac{\lambda}{2} \quad\Rightarrow\quad \lambda = 2 \times \text{node spacing}$$ $$v = f\lambda = f \times 2d_\text{node} \qquad \text{Example: } \lambda = 2(0.84) = 1.68\ \mathrm{m},\quad v = 200 \times 1.68 = 336\ \mathrm{m\,s^{-1}}$$

Exam method: explain stationary wave formation (3 marks)

  1. Incident and reflected waves interfere/superpose.
  2. Constructive interference (waves in phase) → maximum amplitude = antinodes.
  3. Destructive interference (waves in antiphase) → zero amplitude = nodes.
Do not write "superimpose" — OCR ignores this. Use "superpose" or "interfere".

Exam method: why nodes are not exactly zero (3 marks, 2023 Q22b-ii)

  1. Intensity/amplitude of wave decreases with distance from the source.
  2. The reflected wave has a lower amplitude than the incident wave.
  3. Incomplete destructive interference occurs — amplitudes do not fully cancel, so nodes are not exactly zero.
Don't just say "energy is absorbed". You must link it to the amplitude difference causing incomplete cancellation.

5.4 · Young Double-Slit and Interference

Interference conditions and Young's slits $$\text{Constructive: path difference} = n\lambda \quad (n = 0, 1, 2, \ldots)$$ $$\text{Destructive: path difference} = \left(n + \tfrac{1}{2}\right)\lambda$$ $$w = \frac{\lambda D}{d} \quad\Rightarrow\quad \lambda = \frac{wd}{D}$$ $$\text{If given "distance to } n\text{th fringe"}: \quad w = \frac{\text{total distance}}{n}\ \text{first, then substitute.}$$
Young's experiment demonstrates: diffraction, superposition, wave nature of light. It does NOT demonstrate refraction. MCQ asking which is not demonstrated: always refraction.

5.5 · Polarisation — Two MCQ Types

Exam method: distinguish the two polarisation MCQ types

  1. Type 1 — unpolarised source, filter rotated (2023 Q13): after the first filter halves the intensity, rotating it further does not change the intensity. Answer: D (does not change).
  2. Type 2 — polarised source, e.g. laptop screen (2024): at 90° the filter is perpendicular → dark. At 180° it re-aligns → full brightness again. Sequence: normal → dark → normal. Answer: B.
Read carefully: is the source polarised (Type 2) or is only the filter polarising (Type 1)?

5.6 · Refraction and TIR

Snell's law and critical angle $$n_1 \sin\theta_1 = n_2 \sin\theta_2 \qquad \sin\theta_c = \frac{n_2}{n_1}\ (n_1 > n_2)$$ $$n = 1.5 \Rightarrow \theta_c = \arcsin\!\left(\tfrac{1}{1.5}\right) = 41.8^\circ$$ $$\text{Semi-circle MCQ: if incidence angle} > \theta_c \Rightarrow \text{TIR. For } 45^\circ > 41.8^\circ \Rightarrow \text{TIR (Answer A, 2023 Q5).}$$

5.7 · Diffraction and de Broglie

Diffraction and de Broglie wavelength $$\text{Significant diffraction: } \lambda \approx \text{gap width}$$ $$\text{3 cm gap} \Rightarrow \text{microwaves } (\lambda \approx 1\text{–}10\ \mathrm{cm})$$ $$\lambda_\text{de Broglie} = \frac{h}{p} = \frac{h}{mv} \qquad \text{For electron diffraction through graphite: } \lambda \approx d\ (\text{atomic spacing})$$

Section 06 · Spec 6.1

Quantum Physics and the Photoelectric Effect

6.1 · The Photoelectric Effect

Einstein's photoelectric equation $$hf = \phi + KE_\text{max} \qquad h = 6.63\times10^{-34}\ \mathrm{J\,s}$$ $$\text{Threshold: } hf_0 = \phi \qquad \text{Photon energy: } E = hf\ (\text{proportional to frequency, not intensity})$$ $$\text{Work done by accelerating pd: } W = eV \qquad \text{Intensity: } I = \frac{P}{A} = \frac{n_\text{per s} \cdot hf}{A}$$
Increasing intensity (above threshold): more photons/second → more electrons/second → higher current. $KE_\text{max}$ is unchanged because it depends only on $f$.

Exam method: explain why frequency must exceed minimum (3 marks, 2023 Q23b)

  1. Light travels as packets of energy called photons.
  2. An electron is only emitted if a photon's energy is greater than or equal to the work function.
  3. Photon energy is proportional to frequency ($E = hf$), so frequency must exceed the threshold value.
Do not mention intensity here. The issue is energy per photon, not the rate of photon arrival.

Exam method: why current ∝ intensity (3 marks, 2023 Q23c-i)

  1. Intensity $\propto$ rate of incident photons (at above-threshold frequency).
  2. One photon interacts with one electron (one-to-one) → more photons → more electrons released per second.
  3. Current $=$ rate of charge flow $= \dot{n} \cdot e$, so current $\propto$ rate of electron release $\propto$ intensity.

Exam method: calculate intensity from photon rate (4 marks, 2023 Q23c-ii)

  1. $E_\text{photon} = hf = 6.63\times10^{-34} \times 8.2\times10^{15} = 5.44\times10^{-18}\ \mathrm{J}$
  2. $P = E_\text{photon} \times \dot{n} = 5.44\times10^{-18} \times 3.1\times10^{18} = 16.9\ \mathrm{W}$
  3. $I = P/A = 16.9 / 4.9\times10^{-3} = 3.4\times10^3\ \mathrm{W\,m^{-2}}$
Intensity = power per unit area. Power = photons per second × energy per photon. Do not confuse intensity with power.

6.2 · Gold Leaf Electroscope (2024)

Exam method: gold leaf electroscope falls (3 marks)

  1. Electrons on the metal surface absorb photons in a one-to-one interaction.
  2. Electrons are emitted from the metal plate.
  3. The electroscope loses negative charge → leaf becomes less charged → leaf falls.
Do not say the leaf gains positive charge. Electrons come from the metal plate, not the gold leaf itself.

6.3 · Planck Constant Experiment (2024)

LED method for Planck constant $$eV_0 = hf \quad\Rightarrow\quad V_0 = \frac{h}{e} \cdot f$$ $$\text{Plot } V_0 \text{ vs } f:\quad \text{gradient} = \frac{h}{e} \quad\Rightarrow\quad h = \text{gradient} \times e = \text{gradient} \times 1.6\times10^{-19}$$ $$\%\text{uncertainty} = \frac{|h_\text{exp} - h_\text{accepted}|}{h_\text{accepted}} \times 100$$
Cardboard tube: excludes ambient light, making it easier to judge the exact moment the LED starts to emit — reducing the uncertainty in $V_0$.

  • Apply the potential divider formula and compare $V_\text{LED}$ numerically to the switch-on threshold.
  • Explain compressions and rarefactions in a longitudinal wave (three distinct marking points).
  • Explain stationary wave formation and why real nodes are not exactly zero (incomplete destructive interference).
  • Distinguish the two polarisation MCQ types: unpolarised source (constant intensity) vs polarised source (zero at 90°).
